# VehiReID-RAG
VehiReID-RAG is a multimodal framework for vehicle re-identification, combining image captioning with retrieval-augmented generation. It extracts fine-grained visual details, generates descriptive captions, and queries a knowledge base to enable accurate cross-camera matching with grounded, interpretable results.

# How to use
Create a folder and clone the repo into it on you rloval computer/vm. Install the CompCars dataset from the Google Drive in the link https://mmlab.ie.cuhk.edu.hk/datasets/comp_cars/instruction.txt. Only install data files, ignore sv files. Please read the instructions on the stie on how to download.
